# Comment installer le modpack Beecraft

## Installation du launcher

Pour lancer un modpack moddé il faut installer un launcher compatible. J'ai fait le choix d'ATLauncher mais d'autres sont possibles si vous en voulez un autre. Toute la suite concerne AT.

Pour commencer, téléchargez la version qui vous correspond [ici](https://atlauncher.com/downloads) et lancez/installez là.

Une fois lancé vous tombez sur cette page
![](https://github.com/clavierp/BeeCraft/blob/main/res/screenshots/ATLauncher/wellcome%20page.png?raw=true)

Cliquez sur l'onglet "Packs" à droite
![](https://github.com/clavierp/BeeCraft/blob/main/res/screenshots/ATLauncher/packs%20page.png?raw=true)

Cliquez sur "Modrinth" à gauche
![](https://github.com/clavierp/BeeCraft/blob/main/res/screenshots/ATLauncher/modrinth.png?raw=true)

Cliquez sur "Add Manually" en haut
![](https://github.com/clavierp/BeeCraft/blob/main/res/screenshots/ATLauncher/modrinth%20manual%20add.png?raw=true)

Entrez le lien du pack (https://modrinth.com/modpack/beecraft) et cliquez sur ok
Choissisez la bonne version (problement la dernière, voir sur discord si ce n'est pas la bonne) et installez le pack
![](https://github.com/clavierp/BeeCraft/blob/main/res/screenshots/ATLauncher/install%20version%20choice.png?raw=true)

Attendez que le pack s'install tout seul
![](https://github.com/clavierp/BeeCraft/blob/main/res/screenshots/ATLauncher/install%20complete.png?raw=true)

Allez dans l'onglet "Instances" ezt trouvez le pack "BeeCraft" (normalement vous n'en avez qu'un)
![](https://github.com/clavierp/BeeCraft/blob/main/res/screenshots/ATLauncher/instances%20page.png?raw=true)

Cliquez sur "Play"
Féllicitation vous venez de lancer un modpack, il ne reste qu'à se connecter sur le serveur.
